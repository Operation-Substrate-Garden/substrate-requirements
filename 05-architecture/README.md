# Architecture Requirements

This folder contains architectural requirements, system diagrams, and structural constraints for the Operation Substrate Garden project.

Requirements here define how the individual components (backend, web, app) interact and the technical responsibilities of each repository. 
All architectural decisions are rooted in the "Architectural Substrate" philosophy, favoring precision, reliability, and security.

The following documentation is available:
- `system-architecture.md`: High-level system design and component definitions.
