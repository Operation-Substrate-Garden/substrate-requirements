# Tactical Elegance Design System

Tactical Elegance is the design system that provides the "Civilian Accessibility" within Operation Substrate Garden.

## 1. Tonal Layering
- **Definition:** To create depth and structure without using standard 1px borders, we use tonal layering.
- **Layers:** Each layer (e.g., a community card or a topic sidebar) uses a different shade of the background color.
- **Space:** Spacing is the primary tool for creating visual hierarchy, providing "air" and a sense of calm.

## 2. Dual-Font Strategy
- **Manrope (Display):** Used for headlines, community titles, and navigational elements. It provides a modern and approachable feel.
- **Inter (Utility):** Used for all body text, discussion threads, and utility-driven UI elements. It ensures maximum readability and utility.

## 3. Typography and Spacing Scale
- **Typography:** Precise font sizes and line heights are used to maintain a clear hierarchy.
- **Spacing:** A 4px grid-based spacing system ensures consistency throughout the substrate.
- **Contrast:** High-contrast text is used on background tones for maximum accessibility.

## 4. Tone of Voice
- **Professionalism:** All platform copy must be professional and authoritative.
- **Warmth:** Despite its security, the language should be welcoming and helpful.
- **Clarity:** Avoid jargon where possible to remain accessible to both civilian and military users.
